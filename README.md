# Chara-Graph
=======
# App Overview

This application for creating character relationship diagrams that change over time.
This application runs on a local PC.

# Tauri + React + Typescript

キャラクターの動的相関図の作成デスクトップアプリ


## how to build
```bash
npm install
npm run tauri build
```
